Capstone Project Portfolio

1. Appointment Service (Software Design & Engineering)

A Java-based application enhanced with file persistence, search and filter functionality, input validation, and a user-friendly JavaFX GUI. This project demonstrates object-oriented design, data validation, and user interface development using the MVC pattern.

2. Contact Service (Algorithms and Data Structures)

Originally using an ArrayList, this Java application was enhanced by switching to a HashMap for O(1) lookups and adding an autocompleteSearch feature. The project showcases improved algorithmic efficiency and practical application of data structures in contact management scenarios.

3. Grazioso Salvare Dashboard (Databases & Full-Stack)

A full-stack web app built with Node.js, Express, React, and MongoDB. Features include JWT-based authentication, RBAC, advanced MongoDB queries, and dynamic filtering of rescue animal data. Enhanced from a Python dashboard to a scalable, secure web platform.
